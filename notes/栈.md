# 栈

**特点**  
只允许在一端进行插入或删除操作的线性表
LIFO(last in first out)

**栈顶**：允许插入和删除的一端  
**栈底**：不允许插入和删除的一端  

**基本操作**  
初始化  
销毁  
进栈  
出栈  
读栈顶元素  
查栈顶元素
判空

## 输出合法性  

n和不同元素进栈，出栈元素不同排列的个数为  
```math
\frac{1}{(n+1)}C_{2n}^n
```

**合法的输出**  
某个元素出栈之后，在该元素之前入栈的元素出栈顺序是逆序的  
**e.g.**  
由1、2、3、4组成的序列

4123 就是不合法的输出  
如果4最先出栈，那么在4之前入栈的元素出栈顺序就是逆序的，所有4先出栈的合法输出之有4321

## 栈的顺序存储

```
#defien MaxSize 10

//结构体
typedef struct{
    ElemType data[MaxSize];
    int top;
}SqStack;

//判空
void isStackEmpty(SqStack S)
{
    return (S.top == -1)
}

//判满
void isStackFilled(SqStack S)
{
    return (S.top == MaxSize-1)
}

//初始化
void InitStack(SqStack &S)
{
    S.top = -1;
    memset(S.data, 0, sizeof(S.data));
}

//入栈
bool Push(SqStack &S, ElemType e)
{
    if(isStackFilled(S))
    {
        return false;
    }
    S.data[++S.top] = e;
    
    return true;
}

//出栈
bool Pull(SqStack &S, ElemType &e)
{
    if(isStackEmpty(S))
    {
        return false;
    }
    e = S.data[S.top--];
    
    return true;
}

//读栈顶
bool GetTop(SqStack &S, ElemType &e)
{
    if(isStackEmpty(S))
    {
        return false;
    }
    e = S.data[S.top];
    
    return true;
}

//共享栈
typedef struct{
    ElemType data[MaxSize];
    int top0;
    int top1;
}ShStack;

//共享栈初始化
void InitStack(ShStack &S)
{
    S.top0 = -1;
    S.top1 = MaxSize;
    
    memset(S.data, 0, sizeof(S.data));
}

//共享栈判断满
top0 + 1 = top1;
```

## 栈的链式存储  
```
//结构体
typedef struct LNode{
    ElemType data;
    LinkNode *next;
}LNode, *LiStack;

```

### 带头结点
```
//初始化
bool InitStack(LiStack &L)
{
    if(NULL == L)
    {
        return false;
    }
    
    L->next = NULL;
    return true;
}

//判空
bool isStackEmpty(LiStack &L)
{
    if(NULL == L)
    {
        assert(0);
    }
    
    return (L->next == NULL);
}

//入栈
bool PushLiStack(LiStack &L, ElemType e)
{
    if(NULL == L)
    {
        return false;
    }
    
    LNode *s = (LNode*)malloc(sizeof(LNode));
    if(NULL == s)
    {
        return false;
    }
    
    s->data = e;
    s->next = L->next;
    L->next = s;
    
    return true;
}

//出栈
bool PushLiStack(LiStack &L, ElemType &e)
{
    if(NULL == L)
    {
        return false;
    }

    LNode *p = L->next;
    e = L->next->data;
    L->next = L->next->next;
    free(p);
    
    return true;
}

//取栈顶元素
bool PushLiStack(LiStack &L, ElemType &e)
{
    if(NULL == L)
    {
        return false;
    }

    e = L->next->data;
    
    return true;
}
```
